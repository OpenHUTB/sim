# 图件来源与复现说明

本工程中的论文图像已统一存放在 `images/` 目录，并在 LaTeX 正文中通过 `figure` 环境调用，图题与图片绑定。

## 图像来源

- `image5.png` 至 `image19.png`：由原 Word 文档提取，主要为仿真平台界面、不同地形工况仿真结果、风险评分热力图、HMI 告警状态图等。
- `image2.jpeg`、`image3.jpeg`、`image4.jpeg`：签名图像，用于诚信声明与授权书页面。
- `image1.png`：学校标识图像，用于封面与页眉相关元素。

## 复现建议

若后续需要提交可编辑原图，建议按以下方式补充：

1. 对仿真曲线图，优先保留 MATLAB / Python 绘图脚本与原始 CSV 数据；
2. 对平台界面截图，保留仿真程序版本、运行参数与截图来源说明；
3. 对表格数据，优先维护在 LaTeX 表格或 CSV 文件中，而不是截图；
4. 若教师要求“可编辑原图”，应补充生成图片的 `.m`、`.py` 或原始工程文件。

当前版本已保证所有图片以 LaTeX `figure` 环境排版，图注不会与图片分离。
