# 基于高保真模拟器的车辆与飞行器协同控制 LaTeX 工程

本工程根据用户提供的 Word 原文重新整理。

## 编译方式

使用 XeLaTeX 编译：

```bash
xelatex main.tex
xelatex main.tex
```

## 目录结构

- cover/cover.tex：封面
- abstractC/abstractC.tex：中文摘要
- abstractE/abstractE.tex：英文摘要
- body/chapter1.tex ~ body/chapter5.tex：正文五章
- appendix/appendix1.tex、appendix2.tex：附录
- acknowledge/acknowledge.tex：致谢
- ref.tex：参考文献
- pic/：从 Word 中提取的图片

说明：Word 中部分公式和图片来自原文件自动转换，建议最终提交前重点检查公式编号、图片清晰度和图题位置。
