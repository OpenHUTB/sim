# Kinesis 快速开始指南

## 🚀 立即开始 - 3个步骤

---

## 步骤1：下载 SMPL 模型（5分钟）

### 操作流程：

1️⃣ 打开浏览器，访问：
   ```
   https://smpl.is.tue.mpg.de/
   ```

2️⃣ 注册账户
   - 点击 "Sign up"
   - 填写邮箱、密码、姓名
   - 同意条款，提交

3️⃣ 登录
   - 用注册邮箱和密码登录

4️⃣ 下载模型
   - 找到 "Download" 部分
   - 点击 **neutral** 链接（中间那个）
   - 下载文件（约10-20 MB）

5️⃣ 安装
   - 将下载的文件重命名为：`SMPL_NEUTRAL.pkl`
   - 复制到：`D:\兼职\18\Kinesis\data\smpl\`

---

## 步骤2：下载 KIT 数据集（30-60分钟）

### 操作流程：

1️⃣ 打开浏览器，访问：
   ```
   https://amass.is.tue.mpg.de/
   ```

2️⃣ 注册账户
   - 点击 "Sign up"
   - 填写信息
   - 提交注册

3️⃣ 登录
   - 用注册邮箱和密码登录

4️⃣ 找到 KIT 数据集
   - 滚动页面查找 "KIT" 或 "CMU"
   - 数据集描述：CMU Graphics Lab Motion Capture Database
   - 大小：约 2-5 GB

5️⃣ 下载
   - 点击 "Download"
   - 选择 **SMPL-H** 格式
   - 下载 ZIP 文件（约 2-5 GB）

6️⃣ 解压
   - 解压到：`D:\KIT_Data\KIT\`
   - 确保能看到 `.npz` 文件

---

## 步骤3：转换数据（10分钟）

### 打开命令行，运行：

```bash
# 1. 进入Kinesis目录
cd /d "D:\兼职\18\Kinesis"

# 2. 转换KIT数据
python src/utils/convert_kit.py --path "D:\KIT_Data\KIT"
```

**等待转换完成**（约10分钟）

转换完成后，会生成：
- ✅ `data/kit_train_motion_dict.pkl`
- ✅ `data/kit_test_motion_dict.pkl`

## 📞 需要帮助？

如果在任何步骤遇到问题：

### 问题1：网站打不开
- 检查网络连接
- 尝试使用VPN
- 确认URL拼写正确

### 问题2：下载失败
- 尝试使用下载管理器（如IDM）
- 选择非高峰时段
- 确保有足够磁盘空间（至少10GB）

### 问题3：转换失败
- 检查路径是否正确
- 确认NPZ文件存在
- 查看错误信息

### 问题4：运行测试失败
- 先运行验证脚本
- 确认所有文件存在
- 查看终端错误输出
