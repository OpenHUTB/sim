# airsim-swarm-opt

毕设项目：AirSim 多机仿真性能分析

## 背景

在做多无人机集群实验时发现 AirSim 在无人机超过 20 架后帧率严重下降，打算找到原因并改进。

## 计划

- 定位 AirSim 源码里的性能瓶颈
- 设计几种替代方案
- 写测试脚本验证效果

## 开发环境

- AirSim (Midas75 fork)
- Python 3.9
- Windows 11
